API Testing Project – E-commerce System (Postman)

Overview

This project demonstrates API testing of a simulated e-commerce system using Postman.  
It covers core backend functionalities such as authentication, product retrieval, cart management, and order simulation.

The goal of this project is to demonstrate real-world QA skills including API validation, functional testing, and end-to-end workflow testing.

---

APIs Used

This project uses public mock APIs:

- https://reqres.in (Authentication & user management)
- https://fakestoreapi.com (E-commerce data)

---

Tools & Technologies

Postman (API Testing)
JavaScript (Postman Test Scripts)
JSON (Request/Response format)
